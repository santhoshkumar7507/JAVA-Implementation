## 📖 About The Project

Managing student data manually can be a daunting and error-prone task. The **Student Management System** provides a streamlined, text-based interface to perform CRUD (Create, Read, Update, Delete) operations on student profiles. Built entirely in Java, it serves as a lightweight, fast, and reliable tool for educational administrators or developers looking to understand core Java concepts.

### 🌟 Key Features

* **Add New Students**: Easily enroll new students into the system.
* **View Student Records**: Fetch and display details of existing students.
* **Update Information**: Modify student details seamlessly.
* **Remove Students**: Securely delete records from the system.
* **Interactive CLI**: A user-friendly command-line interface that guides you through every step.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have the Java Development Kit (JDK) installed. You can verify this by running:
```bash
java -version
```
*If you don't have Java installed, download it from the [official Oracle website](https://www.oracle.com/java/technologies/downloads/).*

### Installation & Execution

1. **Clone the repository** (or download the source code):
   ```bash
   git clone https://github.com/santhoshkumar7507/JAVA-Implementation.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd JAVA-Implementation
   ```

3. **Compile the code**:
   ```bash
   javac main.java
   ```

4. **Run the application**:
   ```bash
   java main
   ```

## 💻 Usage

Upon launching the application, you will be greeted with an interactive menu:

```text
=======================================
      STUDENT MANAGEMENT SYSTEM
=======================================
1. Add Student
2. View All Students
3. Update Student
4. Delete Student
5. Exit
=======================================
Enter your choice: 
```

Simply input the number corresponding to the action you wish to perform and follow the on-screen prompts!
